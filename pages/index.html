BlinkBin

A simple pastebin service with an open API designed to be simple and accessible for all developers. Create, retrieve, edit, and delete pastes without authentication, protected by rate limiting, proof-of-work, and temporary tokens.

Rate Limits/Protections Overview

  • Challenges: Proof-of-work challenge required for uploading pastes; difficulty ramps up if spam or multiple uploads in a short timeframe are detected. Challenges are limited to 15 issuances/minute and a max of 10 unused, non-expired challenges per IP.
  • Creating pastes: 5/min per IP and adaptive PoW difficulty based on multiple factors
  • Read endpoints: 120/min per IP, with a stricter limit at 25/min per IP for repeated 404s
  • Editing pastes: 3/min per edit token
  • Deleting pastes: 30/min per IP plus a valid edit token to verify ownership
  • Edit/delete fails due to invalid token: 10/min per IP
  • Unknown paths: 45/min per IP

Rate limits are tracked in a SQLite database, persisting across restarts and synced across workers.

Uploading Pastes

  1. Get a proof-of-work challenge: GET /api/challenge
  2. Solve it locally by finding the nonce
  3. Create the paste: POST /api/paste
  4. Save paste_hash and edit_token somewhere safe so you can edit/delete/view your paste later.
import hashlib
import requests

api = "https://bin.blinkl.ink"

# Retrieve a challenge to solve (expiry varies on many factors)
challenge_response = requests.get(f"{api}/api/challenge").json()
challenge, diff = challenge_response["challenge"], challenge_response["difficulty"]

nonce = 0
while True:
    h = hashlib.sha256(f"{challenge}{nonce}".encode()).hexdigest()
    if h.startswith("0" * diff):
        break
    nonce += 1

# Make a POST request to create the paste
r = requests.post(f"{api}/api/paste", json={
    "paste": "hello from python",
    "challenge": challenge,
    "nonce": nonce,
})

Editing Pastes

  1. Send a PUT request to /api/paste/{paste_hash} with your new text and edit token.
import requests

api = "https://bin.blinkl.ink"

# these would be saved from when you uploaded your paste
paste_hash = "abc123XY"
edit_token = "do-not-share-this"

response = requests.put(f"{api}/api/paste/{paste_hash}", json={
    "paste": "updated text",
    "edit_token": edit_token,
})

Retrieving Pastes

Pastes are retrieved using the short paste ID/paste hash. There are three endpoints for this:

  1. JSON: GET /api/paste/{paste_hash}
  2. Raw text: GET /api/paste/{paste_hash}/raw
  3. Rendered page (for sharing with people): GET /{paste_hash}
import requests

api = "https://bin.blinkl.ink"
paste_hash = "abc123XY"

json = requests.get(f"{api}/api/paste/{paste_hash}").json()
raw = requests.get(f"{api}/api/paste/{paste_hash}/raw").text

Deleting Pastes

  1. Send a DELETE request to /api/paste/{paste_hash} with your edit token.
  2. The server will delete the paste if the token matches.
import requests

api = "https://bin.blinkl.ink"

# these would be saved from when you uploaded your paste
paste_hash = "abc123XY"
edit_token = "do-not-share-this"

response = requests.delete(f"{api}/api/paste/{paste_hash}", json={
    "edit_token": edit_token,
})
